Phase 1 - Critical Safety Items

Status: Not Started | Estimated: 2-3 weeks

Planned Tasks:

  1. Exhaust System Replacement
    • Inspect exhaust manifold for cracks
    • Replace complete exhaust system from manifold back
    • Verify no CO intrusion into cabin
  2. Front Driveshaft Replacement
    • Source replacement driveshaft (XJ Cherokee compatible)
    • Replace unsafe amateur-welded shaft
    • Verify U-joint condition and proper installation
  3. Driver's Door Repair
    • Assess weld failure locations
    • Proper welding repair or hinge replacement
    • Verify door alignment and secure attachment
  4. Floor Pan Patch Welding
    • Cut and fit patch panels for rust holes
    • Weld patches securely
    • Apply rust prevention treatment
  5. Brake System Inspection
    • Complete brake system check
    • Replace worn components as needed
    • Verify parking brake function
